# OVH CDS installation and setup

## Prepare the application

```shell
./data/cds-eprotocole-1-installation.sh

# And then execute the export INIT_TOKEN command displayed on the console
```

## Create and administrator user

```shell
./data/cds-eprotocole-2-create-admin.sh

# Enter a strong password when requested and then confirm the default context

# And then try to login on the UI http://localhost:8080
```
